# M17 - Actividad 1 Git - Simulador de Dados

Esta aplicación simula el lanzamiento de dados utilizando Python.

El programa implementa una función que recibe dos parámetros:

- `amount`: cantidad de lanzamientos.
- `sides`: cantidad de caras que tiene el dado.

La función devuelve una lista con los resultados obtenidos al lanzar el dado.

El programa muestra cada resultado con el siguiente formato:

Lanzamiento {número de lanzamiento} número obtenido {resultado lanzamiento}

Entre cada lanzamiento se realiza una pausa de 5 segundos.
